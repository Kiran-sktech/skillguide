# SkillGuide 🎓

SkillGuide is an AI-powered learning mentor for anyone who wants to learn a new skill, solve doubts, or get unstuck. Instead of just giving answers, it guides you step by step — like having a personal mentor available 24/7.

## ✨ Features
- 💬 Multi-chat sidebar — like ChatGPT
- 🧠 AI remembers your conversation history
- 📝 Markdown formatting for clean responses
- ⚡ Fast responses powered by Groq
- 🎯 Mentor-style guidance — not just answers

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **AI:** Groq API (LLaMA 3.3 70B)
- **Tools:** Git, GitHub, dotenv

## 🚀 How to Run Locally

1. Clone the repo
   git clone https://github.com/Kiran-sktech/skillguide.git

2. Install dependencies
   npm install

3. Create a .env file and add your Groq API key
   GROQ_API_KEY=your_key_here

4. Start the server
   npm run dev

5. Open index.html in your browser
